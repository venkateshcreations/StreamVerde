# Assets — Icons

Place SVG icon files here.

Current implementation uses emoji icons via CSS/HTML.
Replace with SVG files as needed and update references in HTML/CSS.

## Recommended icon sets
- [Lucide](https://lucide.dev/) — MIT license, clean line icons
- [Heroicons](https://heroicons.com/) — MIT, Tailwind-compatible
- [Phosphor](https://phosphoricons.com/) — flexible, MIT

## Usage
Reference in HTML:
```html
<img src="../assets/icons/play.svg" alt="Play" width="20" height="20">
```

Or inline SVG directly in HTML for colour control via CSS `currentColor`.
